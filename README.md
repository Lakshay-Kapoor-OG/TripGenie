# TripGenie

TripGenie is an Agentic AI Travel Concierge that generates personalized travel plans based on user preferences.

## Features

- Personalized travel itinerary generation
- Destination intelligence
- Tourist attraction recommendations
- Hidden gems suggestions
- Food recommendations
- Budget planning
- Travel tips and packing suggestions
- Structured Travel Dossier generation

## Technologies Used

- Python
- LangGraph
- LangChain
- Groq LLM
- Gradio

## Agentic AI Concepts Used

- State Management
- Autonomous Planning
- Task Decomposition
- Multi-stage AI Workflow
- Prompt Engineering
- Structured Output Generation

## How It Works

1. User provides travel preferences.
2. The AI processes the input using a state-based workflow.
3. TripGenie generates a personalized Travel Dossier.
4. Destination and place intelligence modules enrich the final output.
5. The results are displayed through an interactive Gradio interface.
